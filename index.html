Flashcards

JavaScript

JavaScript is an object-oriented computer programming language commonly used to create interactive effects within web browsers.

Operators

Operators are used to assign values, compare values, perform arithmetic operations, and more.

Variables

Variables are containers for storing values.

Functions

A function is a block of code written to perform series of instructions.

Methods

In JavaScript, methods are used to manipulate objects. A method is like a function written to perform a task.

Argument

Arguments is an array-like object accessible inside functions that contains the values of the arguments passed to that function.

Parameter

Parameters are variables that you list as part of a function definition. A parameter is a place holder for an argument.

Return

Return statement is used to specify the value that a function should output or "return" back to the caller.

Chaining Methods

Javascript Method Chaining allows us to run multiple methods on the same element within a single statement.

Strings

Strings are series of characters enclosed either with a double or single quote.
